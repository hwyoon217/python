import pandas as pd
df = pd.read_csv("iris.csv")

  • 결측치
    1. 결측치 구하기 : df.isna()

    2. 결측치 제거하기 : df.dropna()

    3. 결측치 치환하기 : df.fillna()


  • 데이터 추출하기
    1. df.iloc[ row , column ] : row와 column에 인덱스 번호를 넣어 원하는 데이터를 추출한다.

    2. df.loc[ row, column ] : row와 column에 라벨 값을 넣어 원하는 데이터를 추출한다. 문자를 사용하여 조건을 넣을 수 있어서 더 편리하다.


  • 데이터 합치기
    1. pd.concat([df_1, df_2])
      2. df_1 옆에 df_2 데이터가 합쳐진다.
      axis=1 옵션을 쓰면 df_2가 df_1 밑에 붙는다.

    2. df_1.merge(df_2, left_on = "", right_on = "" , how= "")
      3. 변수를 기반으로 데이터를 합친다.
      left_on에는 df_1의 변수를, right_on에는 df_2의 변수를 입력하면 된다. how에는 left, right, inner 중 하나를 넣으면 된다.
