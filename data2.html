import pandas as pd
df = pd.read_csv("iris.csv")

  • 인덱스
    1. df.set_index() : 원하는 변수를 인덱스로 설정

    2. df.reset_index() : 인덱스 초기화


  • 시간 변수
    1. df["A"]=pd.to_datetime(df["A"])
      2. 날짜와 시간이 입력되어있으나, 문자형인 df의 A 변수를 시간속성으로 바꾼다.
    2. df["A"].dt.date
      3. A변수 중 날짜부분만 추출
    3. df["A"].dt.hour
      4. A변수 중 시간부분만 추출

      ** 그 외에도 weekday, year 등을 통해 원하는 값을 추출할 수 있다.


  • 데이터 범주화
    • pd.cut(범주화시킬 변수, bins = [ ],labels = [ ], right = True / False )

    bins : 구간
    labels : 각 구간의 이름
    right : True일 땐 각 구간의 범위가 a초과 ~ b이하, False일 땐 a이상 ~ b미만
