import pandas as pd
df = pd.read_csv("iris.csv")

  • 회귀식 입력 시, 모든 column 쓰는 법
    • "종속변수 ~ " + " + ".join(df)


  • 특정 조건을 제외하는 자료 뽑는 법
    • df.loc[ condition, ] : 특정 조건을 만족하는 자료 추출
    df.loc[ ~condition, ] : 특정 조건을 제외하는 자료 추출


  • 문자인 데이터를 숫자로 변환
    1. 값이 2개인 경우
      2. df["gender"] = (df["gender"]=="Male")+0
      이 경우, Male은 1, Female은 0으로 값이 변환된다. 로지스틱 회귀를 사용할 때 유용하다.

    2. 값이 3개 이상인 경우
      3. df["BP_cd"] = df["BP"].replace({"LOW": 0, "NORMAL": 1, "HIGH": 2})
      replace를 사용하여 각각의 문자를 숫자로 변형하면 된다.


  • from scipy.stats import chi2_contingency
    • chi2_contingency(pd.crosstab(df["A"], df["B"]))
    카이제곱 값을 구할 때 사용하며, pd.crosstab을 빼먹지 말자.
